Voice over Internet Protocol
  * a suite of technologies used to carry telephony services over Internet Protocol networks such as the Internet
  * allows voice and video communications to be carried over data packets, using IP addresses

Dial-Peering
  * establishing endpoints of telecommunication by means of routing data packets between telephones connected via Internet or Plain Old Telephone Service (POTS)
    * can be configured for traditional voice connections a la analog telephone connections such as FXS and FXO
    * Foreign Exchange Subscriber provides the dial tone, battery current, and ringing voltage on a telephony line to which an analog device (telephone, fax machine, etc.) connects
    * Foreign Exchange Office receives telephony data through the physical port on an analog telephony device - sends on-hook and off-hook signals to control communications
